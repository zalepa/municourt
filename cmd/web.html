About This Site

This tool visualizes New Jersey municipal court statistics over time. Data is sourced from the NJ Courts public reports and covers metrics such as filings, resolutions, clearance rates, and case backlog across all municipalities and counties in New Jersey.

Use the controls above to select a metric and case type, then add state, county, or municipality-level entities to compare them on the chart. You can add multiple entities to see trends side by side.

Built by Hack Jersey City. Source code available on GitHub.
